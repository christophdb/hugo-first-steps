Combien de fois vous êtes-vous retrouvé devant le supermarché pour constater que vous aviez oublié votre liste de courses à la maison ? Utilisez plutôt une liste de courses numérique à laquelle vous pouvez accéder de n'importe où dans le monde avec votre smartphone ! Vous pouvez même la partager avec les membres de votre famille ou vos colocataires pour qu'ils puissent également ajouter des achats à la liste.

Dans le premier tableau, vous inscrivez tous les produits dont vous avez besoin. Vous pouvez répartir les produits en catégories à l'aide de colonnes de sélection simple ou les trier par nom dans une liste alphabétique. Vous pouvez ainsi facilement trouver ce dont vous avez besoin lorsque vous établissez votre liste de courses. Il suffit de décocher les cases pour que les produits s'affichent dans la vue filtrée "Liste de courses". Dans cette vue, que vous utilisez lorsque vous faites vos courses, les entrées disparaissent à nouveau lorsque vous décochez les cases. Une fois que vous avez tout coché sur votre liste d'achats, cette vue est vide.

Vous pouvez associer les produits aussi bien aux recettes (tableau 2) pour lesquelles vous en avez besoin qu'aux magasins (tableau 3) dans lesquels vous préférez les acheter. Le plug-in de carte vous permet en outre d'afficher l'emplacement des magasins et de planifier votre itinéraire d'achat. Il est judicieux de classer vos achats dans un certain ordre - en fonction de l'endroit où se trouvent les produits. Vous vous déplacez ainsi efficacement dans les magasins et avez tout acheté en un rien de temps.

Découvrez les avantages des listes d'achats dans notrearticle de blog.
