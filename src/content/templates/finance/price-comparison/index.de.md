Sie möchten demnächst eine Anschaffung tätigen und suchen nach einer Möglichkeit verschiedene Preise miteinander zu vergleichen? Mit SeaTable behalten Sie den Überblick, egal wie viele Produkte oder wie viele Preise Sie miteinander vergleichen wollen.

Durch die automatische farbliche Hervorhebung behalten Sie immer den Überblick. Die günstigsten Preise haben einen hellen Farbton, während die teuersten Händler einen dunklen Farbton erhalten. Durch die verschiedenen Ansichten können Sie genau auf die Produkte oder Eigenschaften filtern, auf die es ankommt.

In einer zweiten Tabelle erfassen Sie zusätzliche Details und mithilfe von Statistiken können Sie die Ergebnisse zusätzlich visualisieren.

​

## Interaktives Template

Scrollen Sie durch unser interaktiv eingebettetes Template oder lesen Sie die Beschreibung, indem Sie auf das i-Symbol hinter dem Vorlagennamen klicken. So bekommen Sie ein besseres Gefühl für die Funktionen. Falls Sie Fragen zu SeaTable haben, legen wir Ihnen unserenHilfebereichans Herz.
