Para organizar um projeto inicialmente abstrato em elementos planeáveis, pode utilizar um plano de estrutura do trabalho para dividir o processo em subtarefas e pacotes de trabalho. Antes de a sua equipa elaborar pacotes de trabalho individuais, deve definir as subtarefas em que o seu projeto está dividido. É possível organizar estes grupos de nível superior de acordo com as fases, funções ou objectos.

Quando o objetivo e as subtarefas do seu projeto estiverem claros, é altura de definir os pacotes de trabalho. Descreva a tarefa, adicione uma pessoa responsável, um local, uma hora de início e um prazo. Pode também anotar as suas subtarefas e horas de trabalho. Uma vez que as fases do projeto (Quadro 2) estão ligadas aos pacotes de trabalho (Quadro 1), pode atribuir cada pacote de trabalho a uma fase e assim obter uma visão geral do progresso do projeto e das horas trabalhadas.

Utilize diferentes vistas que filtram as tarefas de cada responsável para apresentar listas de tarefas personalizadas para cada membro da equipa. Tire partido do calendário e do plugin de cronologia para mapear períodos de tempo e controlar os prazos.
