Este modelo foi desenvolvido para um departamento de comunicação fictício de uma grande empresa. Neste departamento, todas as actividades de marketing e comunicação têm de ser planeadas muito cedo, muitas vezes com um ano de antecedência.

Para manter um registo de todas as acções planeadas para atingir os objectivos de marketing/comunicação, existe um quadro neste modelo com um calendário completo para o ano de 2022. Cada dia é aqui enumerado para criar um plano para todo o ano. O segundo quadro contém todos os objectivos estratégicos que têm de ser alcançados. Está ligada ao plano anual.

Pode encontrar instruções sobre como criar um plano de ação para um ano inteiro no nossoblogue.

## Modelo interativo

Percorra o nosso modelo incorporado de forma interactiva ou leia a descrição clicando no símbolo i a seguir ao nome do modelo. Isto dar-lhe-á uma melhor noção das funções. A nossasecção de ajudaé o melhor lugar para ir se tiver alguma dúvida.
