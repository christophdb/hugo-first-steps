Este modelo é uma solução pronta a usar para capturar dados de hóspedes em restaurantes, bares e cafés durante a pandemia de Corona. Sem papel, sem canetas, sem complicações - este modelo contém tudo o que é necessário para digitalizar o registo de convidados.

A chave para este modelo é o formulário de registo de convidados. Os convidados digitalizam o código QR do formulário com a câmara do seu smartphone e depois introduzem as suas informações de contacto no seu próprio dispositivo. O modelo também inclui três vistas pré-configuradas: Todas as Inscrições, Hoje e Para Apagar. Este último filtra todos os registos com mais de um mês. Devem ser apagados para cumprir os regulamentos de protecção de dados.
