Ce modèle est une solution prête à l'emploi pour l'enregistrement des données des clients dans les restaurants, les bars et les cafés pendant la pandémie de Corona. Pas de papier, pas de stylos, pas de soucis - ce modèle contient tout ce dont vous avez besoin pour numériser l'enregistrement des invités.

La clé de ce modèle est le formulaire d'enregistrement des invités. Les invités scannent le code QR du formulaire avec l'appareil photo de leur smartphone et saisissent ensuite leurs coordonnées sur leur propre appareil. Le modèle contient également trois vues préconfigurées : Tous les enregistrements, Aujourd'hui et À supprimer. Ce dernier filtre toutes les inscriptions datant de plus d'un mois. Elles devraient être supprimées afin de respecter les règles de confidentialité.

Consultez notrearticle de blogpour découvrir comment la saisie numérique des contacts est possible de manière hygiénique, flexible et simple.
