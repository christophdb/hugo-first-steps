Placez le formulaire web ou le code QR généré par SeaTable sur votre site web et donnez ainsi à vos visiteurs la possibilité de s'inscrire à votre newsletter. Lors de l'inscription, l'intéressé peut choisir lui-même ses centres d'intérêt.

​

## Modèle interactif

Faites défiler notre modèle intégré de manière interactive ou lisez la description en cliquant sur l'icône i derrière le nom du modèle. Vous aurez ainsi une meilleure idée des fonctions. Si vous avez des questions, nous vous conseillons de consulter notrerubrique d'aide.
