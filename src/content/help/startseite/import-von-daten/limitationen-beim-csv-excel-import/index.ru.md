SeaTable имеет мощную функцию импорта, чтобы максимально упростить процесс **импорта записей данных из файлов XLSX или CSV**. В большинстве случаев импорт проходит без каких-либо затруднений. Если вы попытаетесь импортировать слишком много записей одновременно, SeaTable предупредит вас об установленном ограничении. В то время как на собственных серверах SeaTable вы можете свободно выбирать **ограничения на размер файла и количество строк**, SeaTable Cloud накладывает на вас определенные ограничения.

## Максимальный размер файла и количество строк для SeaTable Cloud

Если вы импортируете файлы Excel, SeaTable поддерживает их для файлов размером до **100 МБ** и до **50 000 строк**. Благодаря таким щедрым ограничениям у вас никогда не возникнет проблем с размером файла. Если файл больше или содержит больше строк, вам придется разделить его на два **файла Excel** и выполнить импорт в два этапа.

## Встроенная графика и файлы

Импорт графики и файлов через файл XLSX **невозможен**. Для такого импорта необходимо использовать API SeaTable, с помощью которого можно без проблем импортировать изображения и файлы.

## Различные форматы файлов

Функция импорта SeaTable была протестирована с файлами из **LibreOffice**, а также с текущими файлами из **Microsoft Excel**. Другие программы Office (например, от компании Softmaker) активно **не** тестируются, что может привести к отклонениям в результатах импорта. Если сомневаетесь, используйте Microsoft Excel или бесплатный LibreOffice для получения наилучших результатов.
