Puede crear nuevas bases en SeaTable no sólo vacías o a partir de una plantilla, sino también a partir de un archivo DTABLE. DTABLE es el [formato de archivo]({{< relref "help/startseite/import-von-daten/dtable-dateiformat" >}}) interno utilizado para exportar y guardar bases desde SeaTable. Aquí puede aprender a crear una base con un archivo DTABLE:

## Creación de una base a partir de un archivo DTABLE

1. Cambie a la página de **inicio** de SeaTable.
2. Dependiendo de si desea crear una nueva base a partir de un archivo DTABLE en el área de trabajo **Mis bases** o en un **grupo** específico, haga clic en **Añadir una base o carpeta** en el área correspondiente.
3. Seleccione **Importar base desde archivo (XLSX, CSV o DTABLE)**.
4. En la vista general de archivos que se abre, seleccione el **archivo DTABLE** deseado de su dispositivo a partir del cual desea crear la base.
5. Confirme la importación **haciendo doble clic** en el fichero correspondiente o pulsando **Seleccionar** o **Abrir**.

## Problemas de importación por falta de versiones o permisos

Entre otras cosas, con las **automatizaciones** y los **conjuntos de datos compartidos**, pueden surgir problemas si el exportador de un archivo DTABLE tiene determinados permisos o autorizaciones de los que carece otro miembro del equipo.

Por ejemplo, si un usuario importa una base sin los **permisos** y **aprobaciones** adecuados, **no** podrá acceder a los registros compartidos ni realizar automatizaciones.

{{< warning  headline="Consejo"  text="Asegúrese de que otros usuarios tienen todos los permisos necesarios antes de proporcionarles un archivo DTABLE para importar una base." />}}
