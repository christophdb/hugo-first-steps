Dans l'article suivant, vous trouverez des instructions sur la manière d'importer des données de Google Sheets dans SeaTable.

## Importation de données de Google Sheets dans SeaTable

1. Pour pouvoir importer des fichiers de Google Sheets dans SeaTable, enregistrez-les d'abord en tant que **CSV-** ou **Fichier Excel**.

    ![Enregistrer Google Sheet au format CSV ou Excel](https://seatable.io/wp-content/uploads/2022/10/google-sheets-export-as-xls-csv.png)

2. Vous pouvez ensuite **importer** les fichiers dans SeaTable comme d'habitude.
3. Eine detaillierte **Anleitung** zum Importieren der Dateien finden Sie im jeweiligen Übersichtsartikel zum [Excel-](https://seatable.io/fr/docs/import-von-daten/import-von-excel-dateien-in-seatable/) bzw. [CSV-Import](https://seatable.io/fr/?post_type=docs&p=16604).
