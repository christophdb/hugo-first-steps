Cada grupo en SeaTable tiene un llamado **ID de espacio de trabajo**. Necesita este ID para poder realizar algunas [llamadas a la API](https://api.seatable.io). Aquí puede encontrar cómo determinar el ID del espacio de trabajo de un grupo a través del navegador.

## Determinar el ID de espacio de trabajo de un grupo

1. Cambie a la página de **inicio** de SeaTable.
2. Haga clic en **cualquier base del grupo** cuyo **ID de espacio de trabajo** desee determinar.
3. Cuando la base seleccionada se ha abierto, puede leer el **ID del espacio de trabajo** en la parte superior de la **URL de** la página.
