Pour personnaliser votre espace de travail à votre guise, vous avez la possibilité de modifier l'ordre dans lequel vos groupes apparaissent sur la page d'accueil de SeaTable. Découvrez ici comment modifier l'ordre des groupes :

## Modifier l'ordre des groupes :

![Modifier l'ordre des groupes](images/change-the-order-of-the-groups-3.gif)

1. Passez à la **page d'accueil** de SeaTable.
2. Sur le côté gauche, cliquez sur l'**icône en forme de triangle** à droite de l'espace de travail **Bases**.
3. Dans le menu déroulant qui s'ouvre, vous voyez maintenant vos **groupes** ainsi que les espaces de travail **Mes bases**, **Favoris** et **Partagé avec moi**.
4. Pour modifier l'ordre des groupes, il suffit de **glisser-déposer** n'importe quel groupe à la position souhaitée. Pour ce faire, utilisez les **six points** qui se trouvent **à droite de chaque groupe**.

{{< warning  headline="Remarque importante"  text="Le _nouvel_ ordre de vos groupes ne sera actif que si vous _rechargez_ la page d'accueil de SeaTable dans votre navigateur ou si vous appuyez sur _F5_ ." />}}
