Para facilitar o acompanhamento das suas bases no SeaTable, tem a opção de atribuir uma cor e um ícone a uma base quando a cria. Pode alterar posteriormente tanto a cor como o ícone. Aqui pode aprender como personalizar o aspecto de uma base:

![Personalizar a aparência de uma base (ícone e cor)](images/change-the-look-of-a-base-3.gif)

1. Mudar para a página **inicial** da SeaTable.
2. Mova o rato sobre a **base** cuja aparência deseja **ajustar**.
3. Para editar, clicar no **símbolo do lápis** na área da direita, que está directamente à esquerda dos **três pontos**.
4. Seleccionar uma nova **cor** e **ícone** para a base correspondente.
5. **Confirmar** as alterações premindo a **tecla Enter**.
   {{< warning  headline="Nota importante"  text="Se quiser editar o aspecto de uma base que pertence a um grupo, deve ser o **proprietário** ou **administrador do** grupo. Os **membros** simples **não** estão autorizados a realizar esta acção." />}}
