Si, à un moment donné, vous n'avez plus besoin des **jeux de données communs** que vous avez créés, vous pouvez facilement les supprimer à tout moment depuis la page d'accueil.

## Suppression d'un jeu de données commun

![Suppression d'un jeu de données commun](images/screnshot-common-datasets-3.png)

1. Ouvrez la **page d'accueil** de SeaTable.
2. Cliquez sur l'onglet **jeu de données commun**.
3. Sélectionnez le jeu de données commun que vous souhaitez **supprimer**.
4. Cliquez sur les **trois points**, puis sur **Supprimer**.
5. Confirmez à nouveau en cliquant sur **Supprimer**.

![Suppression d'un jeu de données commun.](images/delete-a-common-dataset-1.png)

## Effets de la suppression

- Il n'y a **plus de possibilité de synchronisation** pour tous les **tableaux** qui utilisent les données du jeu de données commun supprimé.
- Toutefois, les **données du jeu de données commun** qui se trouvaient jusqu'à présent dans les tableaux correspondants sont **conservées**.
