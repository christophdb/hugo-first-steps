In jeder zu einer Gruppe gehörenden Base können Sie eine Tabellenansicht als **gemeinsamen Datensatz** anlegen. Voraussetzung dafür ist, dass Sie Eigentümer oder Administrator der Gruppe (und nicht nur Mitglied) sind. Den gemeinsamen Datensatz können Sie anschließend inklusive seiner spezifischen Filter- und Sortierbedingungen als neue Tabelle in eine beliebige andere Base der Gruppe importieren.

{{< warning  type="warning" headline="Keine gemeinsamen Datensätze in privaten Bases"  text="Gemeinsame Datensätze stehen nur in **Bases einer Gruppe** zur Verfügung. In einer privaten Base ist die Funktion zum Anlegen eines gemeinsamen Datensatzes ausgegraut." />}}

![](images/common-dataset-only-in-group.png)

## Anlegen eines gemeinsamen Datensatzes

![Anlegen eines gemeinsamen Datensatzes](images/create-a-common-dataset-5.png) ![Anlegen eines gemeinsamen Datensatzes](images/create-a-common-dataset-2-1.png)

1. Öffnen Sie eine beliebige **Tabellenansicht** in einer Base, die zu einer Ihrer Gruppen gehört.
2. Klicken Sie auf die **drei Punkte**.
3. Klicken Sie auf **Als gemeinsamen Datensatz veröffentlichen**.
4. **Benennen** Sie den gemeinsamen Datensatz.
5. Bestätigen Sie mit **Abschicken**.

## Übersicht aller gemeinsamen Datensätze

Sie finden alle von Ihnen erstellten und an Sie freigegebenen gemeinsamen Datensätze auf der Startseite unter **Gemeinsamer Datensatz**.

![Übersicht gemeinsame Datensätze](images/overview-common-datasets-5.png)
