En cada base perteneciente a un grupo, puede crear una vista de tabla como **registro común**. El requisito para ello es que seas el propietario o administrador del grupo (y no sólo un miembro). A continuación, puede importar el conjunto de datos compartidos, incluidas sus condiciones específicas de filtrado y ordenación, como una nueva tabla en cualquier otra base del grupo.

{{< warning headline="Sin registros de datos compartidos en bases privadas" text="Los registros de datos compartidos sólo están disponibles en **Bases de un grupo** está disponible. En una base privada, la función para crear un registro de datos compartido aparece atenuada." />}}

## Creación de un conjunto de datos común

![Creación de un conjunto de datos común](https://seatable.io/wp-content/uploads/2022/11/create-a-common-dataset-5.png)  
![Creación de un conjunto de datos común](https://seatable.io/wp-content/uploads/2022/11/create-a-common-dataset-2-1.png)

1. Abra cualquier **vista de tabla** en una Base que pertenezca a uno de sus grupos.
2. Haz clic en los **tres puntos**.
3. Haga clic en **Publicar como registro compartido**.
4. **Nombra** el registro común.
5. Confirme con **Enviar**.

## Resumen de todos los conjuntos de datos comunes

Encontrará todos los registros de datos compartidos que haya creado y le hayan sido comunicados en la página de inicio, en **Registro de datos compartidos**.
