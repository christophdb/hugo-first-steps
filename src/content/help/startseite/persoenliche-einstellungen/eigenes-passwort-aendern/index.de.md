Sie möchten Ihr eigenes **Passwort** ändern? Kein Problem! In den **persönlichen Einstellungen** Ihres Kontos ist dies in nur wenigen Schritten möglich.

## So ändern Sie Ihr Passwort

1. Wechseln Sie zur **Startseite** von SeaTable.
2. Klicken Sie auf Ihr **Avatarbild** in der oberen rechten Ecke, um das Drop-down-Menü zu öffnen.
   ![Persönliche Einstellungen über Drop-down-Menü öffnen](images/Persoenliche-Einstellungen-ueber-Drop-down-Menue-oeffnen.png)4. Klicken Sie auf **Persönliche Einstellungen**.
3. Klicken Sie im Abschnitt "Passwort" auf **Aktualisieren**.
4. Füllen Sie die angezeigten Felder aus:
    - **Aktuelles Passwort**
    - **Neues Passwort**
    - **Passwort bestätigen**

![Passwort aktualisieren](images/Passwort-aktualisieren.png)

Um sich ein Passwort im **Klartext** anzeigen zu lassen, klicken Sie rechts daneben auf das **Augensymbol**. Zudem bekommen Sie automatisch die **Stärke** Ihres neuen Passworts angezeigt.

![Stärke des Passworts beurteilen](images/Staerke-des-Passworts-beurteilen.png)

16. Bestätigen Sie die Änderung Ihres Passworts mit **Aktualisieren**.
