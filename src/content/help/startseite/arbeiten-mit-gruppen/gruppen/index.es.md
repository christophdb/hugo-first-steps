**Los grupos** son especialmente importantes para la colaboración en tu equipo. Reúnen a miembros de equipos implicados en proyectos conjuntos, permiten compartir fácilmente datos en bases comunes y trasladan las funciones de gestión de usuarios directamente al grupo. Cuanto mayor sea su equipo, más a menudo deberá utilizar los grupos.

¿Quiere utilizar los grupos para la cooperación en su equipo? Este artículo explica todo lo que necesita saber.

## Funciones del grupo

Un grupo de SeaTable puede tener un número ilimitado de miembros. Sin embargo, los miembros del grupo deben formar parte de su equipo.

Cada miembro del grupo tiene uno de los tres papeles: Propietario, administrador o miembro. El usuario que crea un grupo es el propietario del mismo. Puede añadir más miembros y administradores de grupo y cambiar los permisos posteriormente. Un grupo siempre tiene un solo propietario, pero la propiedad puede ser transferida. Un administrador de grupo tiene amplias posibilidades de gestión del grupo y puede nombrar a otros administradores. Sin embargo, un administrador de grupo no puede eliminar un grupo ni transferir su propiedad.

| Acción                             | Propietario | Administrador | Miembro |
| ---------------------------------- | ----------- | ------------- | ------- |
| Cambiar el nombre del grupo        | Sí          | No            | No      |
| Grupo de transferencia             | Sí          | No            | No      |
| Borrar el grupo                    | Sí          | No            | No      |
| Dejar el grupo                     | No          | Sí            | Sí      |
| Gestionar los miembros             | Sí          | Sí            | No      |
| Gestionar la papelera de reciclaje | Sí          | Sí            | No      |

Cada miembro del grupo tiene acceso de escritura a todas las bases del grupo, es decir, a las bases que pertenecen a un grupo. Un miembro puede personalizar las tablas de una base de grupo y crear, modificar y eliminar entradas. El derecho de acceso de un miembro del grupo también incluye el derecho a crear formularios web e instalar plugins.

| Acción                                 | Propietario | Administrador | Miembro |
| -------------------------------------- | ----------- | ------------- | ------- |
| Crear la base del grupo                | Sí          | Sí            | No      |
| Cambiar el nombre de la base del grupo | Sí          | Sí            | No      |
| Copiar la base del grupo               | Sí          | Sí            | Sí      |
| Base del grupo de exportación          | Sí          | Sí            | Sí      |
| Borrar la base del grupo               | Sí          | Sí            | No      |
| Crear acciones de usuario/grupo        | Sí          | Sí            | No      |
| Crear enlace de invitación             | Sí          | Sí            | No      |
| Crear enlaces externos                 | Sí          | Sí            | No      |
| Restaurar instantánea                  | Sí          | Sí            | Sí      |
