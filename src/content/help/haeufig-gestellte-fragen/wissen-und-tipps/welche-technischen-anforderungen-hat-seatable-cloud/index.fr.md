Vous êtes intéressé par SeaTable Cloud, mais vous n'êtes pas encore sûr des **exigences techniques** à remplir ? Pas de problème, vous trouverez dans cet article toutes les informations techniques que vous devez connaître au préalable concernant le navigateur, le système d'exploitation, etc.

## Exigences techniques

- SeaTable Cloud n'a **aucune exigence** en matière de RAM, de CPU ou autre.
- Vous n'avez pas besoin d'un système d'exploitation spécifique sur votre appareil. SeaTable Cloud est pris en charge par **tous les systèmes d'exploitation**.
- Vous avez seulement besoin d'un **navigateur** actuel avec **JavaScript** activé - nous recommandons **Chrome**. D'autres navigateurs, par exemple **Firefox**, peuvent présenter de légères différences dans l'affichage et le fonctionnement.
