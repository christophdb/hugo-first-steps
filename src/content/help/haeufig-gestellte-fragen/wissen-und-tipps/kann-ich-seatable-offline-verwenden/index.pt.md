O **SeaTable Cloud** é um serviço a que acede através da Internet. O acesso sem uma ligação à Internet não é possível.

Em alternativa, pode operar o seu **próprio servidor SeaTable**. Pode aceder a este através da sua rede local e sem acesso à Internet.

## SeaTable Cloud é um produto SaaS

O SeaTable Cloud é e continua a ser um produto de software como um serviço. Isto significa que o SeaTable Cloud é uma plataforma web a que acede com o seu navegador através da Internet. O acesso off-line ao SeaTable Cloud não é actualmente possível.

No entanto, existe, naturalmente, a possibilidade de exportar os seus dados para aceder aos mesmos off-line. A SeaTable oferece-lhe a possibilidade de exportar conjuntos de dados via CSV ou Excel. Em alternativa, pode utilizar o SeaTable API ou outra integração para exportar os seus dados para outro sistema.

{{< warning headline="Preste atenção ao que é exportado" text="Uma exportação de dados via CSV ou Excel não inclui comentários, ficheiros, automatizações e descrições de linhas e tabelas. A reconciliação com os dados exportados pode colocar certos desafios." />}}

## Pode utilizar o seu próprio servidor SeaTable sem a Internet

A situação é diferente se gerir o seu próprio servidor SeaTable na sua rede local. Um SeaTable auto-hospedado não requer acesso à Internet. Neste caso, pode também aceder à sua plataforma e aos seus dados offline através da rede local.
