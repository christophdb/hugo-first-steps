SeaTable vous permet d'ajouter de la couleur à votre travail quotidien, surtout par rapport aux bases de données monotones et aux feuilles de calcul tristes. Avec l ['abonnement Enterprise]({{< relref "pages/prices" >}}), il est même possible d'insérer **vos propres couleurs personnalisées** par code hexadécimal dans la palette de couleurs que vous rencontrez à différents endroits d'une base.

Vous pouvez par exemple colorer les [options de sélection]({{< relref "help/base-editor/spaltentypen/die-einfachauswahl-spalte" >}}), [les cellules]({{< relref "help/base-editor/ansichten/einfaerben-von-zellen" >}}) et [les boutons]({{< relref "help/base-editor/spaltentypen/die-schaltflaeche" >}}) en fonction de votre identité visuelle ou laisser libre cours à votre créativité en matière de couleurs.

{{< warning  headline="Remarque importante"  text="**Seuls** les **propriétaires** et **les administrateurs** peuvent modifier les paramètres de couleur d'une base appartenant à un groupe. Les simples membres du groupe ne voient **pas** cette option." />}}

## Ajouter ses propres couleurs

![Ajouter une couleur personnalisée à votre base](images/add-a-individual-colour-to-your-base.gif)

1. Cliquez sur les **trois points** dans l'en-tête de la base.
2. Allez dans **Paramètres**.
3. Sélectionnez les **paramètres de couleur**.
4. Cliquez sur le **symbole \[+\]** sous l'onglet _Mes couleurs_.
5. Pour la **couleur d'arrière-plan**, choisissez n'importe quelle couleur dans le **sélecteur de couleurs** _ou_ ajoutez une couleur définie par l'utilisateur par **code hexadécimal** dans la palette de couleurs.
6. Choisissez le noir ou le blanc comme **couleur de police**.
7. Confirmez en cliquant sur **Envoyer**, puis cliquez à nouveau sur **Envoyer** pour enregistrer la nouvelle couleur ajoutée.

## Modifier les couleurs

Les couleurs ajoutées à une base peuvent être **adaptées** à tout moment par le même moyen.

![Édition d'une couleur personnalisée ajoutée à votre base](images/edit-individual-colours-added-to-your-base.gif)

1. Cliquez sur les **trois points** dans l'en-tête de la base.
2. Allez dans **Paramètres**.
3. Sélectionnez les **paramètres de couleur**.
4. Cliquez sur **Modifier** sous l'onglet _Mes couleurs_.
5. Cliquez sur la **couleur** que vous souhaitez modifier.
6. Choisissez une **nouvelle couleur de fond et de police**.
7. Confirmez les modifications en cliquant sur **Envoyer**, puis cliquez à nouveau sur **Envoyer** pour enregistrer les modifications.

## Effacer les couleurs

Les couleurs ajoutées à une base peuvent être **supprimées** à tout moment en suivant la même procédure.

![Supprimer une couleur individuelle ajoutée à votre base](images/delete-individual-colours-added-to-your-base.gif)

1. Cliquez sur les **trois points** dans l'en-tête de la base.
2. Allez dans **Paramètres**.
3. Sélectionnez les **paramètres de couleur**.
4. Cliquez sur **Modifier** sous l'onglet _Mes couleurs_.
5. Cliquez sur le **symbole x** à côté de la couleur que vous souhaitez supprimer.
6. Confirmez la suppression en cliquant sur **Envoyer**.

{{< warning  headline="Remarque" >}}

Actuellement, il n'est **pas** possible dans SeaTable Cloud de définir une couleur personnalisée pour une base (en-tête de base, icône de base sur la page d'accueil) ou des [formulaires]({{< relref "help/base-editor/webformulare/webformulare" >}}). Ici, seules les **12 couleurs** prédéfinies sont disponibles.

{{< /warning >}}
