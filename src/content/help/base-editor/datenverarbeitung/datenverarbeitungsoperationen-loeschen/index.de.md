Mithilfe der Datenverarbeitungsfunktion können Sie diverse **Operationen** durchführen. Dabei können Sie sowohl mathematische Berechnungen über alle Werte einer Spalte hinweg vornehmen als auch Werte miteinander in Beziehung setzen, wenn Einträge in zwei verschiedenen Tabellen identisch sind.

Bereits angelegte Datenverarbeitungsoperationen können Sie jederzeit über die Ansichtsoptionen in Ihren Tabellen löschen.

## Löschen einer Datenverarbeitungsoperation

![Klicken Sie auf Datenverarbeitung](images/create-an-data-processing-action-1.jpg)

1. Öffnen Sie eine beliebige **Tabelle**, in der Sie eine Datenverarbeitungsoperation löschen möchten.
2. Klicken Sie auf die **drei Punkte** in den Ansichtsoptionen.
3. Klicken Sie auf **Datenverarbeitung**.
4. Fahren Sie mit der Maus über die angelegte Datenverarbeitungsoperation, die Sie löschen möchten, und klicken Sie auf das **Papierkorb-Symbol**.
   ![Löschen einer Datenverarbeitungsaktion](images/delete-data-processing-actions.jpg)

## Folgen der Löschung

Wenn Sie eine angelegte Datenverarbeitungsoperation löschen, wird diese **unwiderruflich gelöscht** und ist demzufolge **nicht wiederherstellbar**. Die berechneten bzw. miteinander in Beziehung gesetzten Werte in der **Ergebnisspalte** Ihrer Tabelle bleiben jedoch weiterhin erhalten.

Eine gelöschte Datenverarbeitungsoperation können Sie natürlich **nicht** mehr ausführen, d.h. im Nachhinein der Quellspalte hinzugefügte Einträge können Sie **nicht** mehr berücksichtigen. Falls Sie zu einem späteren Zeitpunkt wieder mit der Datenverarbeitungsoperation in Ihrer Tabelle arbeiten möchten, können Sie diese aber jederzeit [neu anlegen]({{< relref "help/base-editor/datenverarbeitung/datenverarbeitungsoperationen-anlegen" >}}).
