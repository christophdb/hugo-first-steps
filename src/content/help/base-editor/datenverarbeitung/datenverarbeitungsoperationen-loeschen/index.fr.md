La fonction de traitement des données vous permet d'**effectuer** diverses **opérations**. Vous pouvez effectuer des calculs mathématiques sur toutes les valeurs d'une colonne ou établir une relation entre les valeurs si les entrées de deux tableaux différents sont identiques.

Vous pouvez à tout moment supprimer les opérations de traitement des données déjà créées en utilisant les options d'affichage dans vos tableaux.

## Suppression d'une opération de traitement des données

![Cliquez sur Traitement des données](images/create-an-data-processing-action-1.jpg)

1. Ouvrez n'importe quelle **table** dans laquelle vous souhaitez supprimer une opération de traitement des données.
2. Cliquez sur les **trois points** dans les options d'affichage.
3. Cliquez sur **Traitement des données**.
4. Passez la souris sur l'opération de traitement des données créée que vous souhaitez supprimer et cliquez sur l'**icône de la corbeille**.

![Suppression d'une action de traitement des données](images/delete-data-processing-actions.jpg)

## Conséquences de la suppression

Si vous supprimez une opération de traitement des données que vous avez créée, celle-ci est **irrémédiablement supprimée** et ne **peut** donc **plus être restaurée**. Les valeurs calculées ou mises en relation dans la **colonne des résultats** de votre tableau sont toutefois conservées.

Vous ne pouvez évidemment **plus** exécuter une opération de traitement des données supprimée, c'est-à-dire que vous ne pouvez **plus** tenir compte des entrées ajoutées ultérieurement à la colonne source. Si vous souhaitez à nouveau utiliser l'opération de traitement des données dans votre tableau à une date ultérieure, vous pouvez toutefois la [recréer à](https://seatable.io/fr/docs/datenverarbeitung/datenverarbeitungsoperationen-anlegen/) tout moment.
