**Динамические элементы** изменяются в соответствии с определенными критериями, но не зависят от записей в вашей таблице. Эти поля **предварительно заполнены** и **автоматически** корректируют свои записи.

### Существуют следующие динамические элементы:

- Текущая дата
- Имя шаблона
- Текущий пользователь

**Настройки**, которые можно выполнить для динамических элементов, такие же, как и для статических текстовых элементов. Поэтому подробные объяснения всех параметров настройки вы можете прочитать в статье [Статический текст](https://seatable.io/ru/docs/seitendesign-plugin/statischer-text/).

При необходимости прочитайте статью [Инструкции к плагину дизайна страниц]({{< relref "help/base-editor/plugins/anleitung-zum-seitendesign-plugin" >}}), чтобы узнать, какие еще элементы и функции предлагает **плагин дизайна страниц**.

## Текущая дата

**Текущая дата** автоматически настраивается на сегодняшнюю без необходимости устанавливать ее вручную. Это позволяет, например, создавать индивидуальные сопроводительные письма, не допуская ошибки в виде устаревшей даты.

Единственной дополнительной настройкой по сравнению со статическими текстовыми элементами является **настройка формата**. Здесь вы можете выбрать различные региональные обозначения для дат.

![формат даты динамического элемента](images/Dynamisches-Element-Datum-Format.png)

## Имя шаблона

**Имя шаблона**, т.е. дизайна страницы, который вы редактируете в данный момент, автоматически корректируется. Если вы изменили название шаблона, изменение в этом поле будет автоматически принято.

![Динамический элемент Имя шаблона](images/dynamisches-Element-Vorlagenname.gif)

## Текущий пользователь

Ваше собственное имя появляется в поле **Текущий пользователь**, если в данный момент вы вносите изменения в шаблон через свою учетную запись. Как только другой **член команды** вызывает шаблон в плагине дизайна страницы, его имя отображается.

![Динамический элемент текущего пользователя](images/dynamisches-Element-Aktueller-nutzer.gif)
