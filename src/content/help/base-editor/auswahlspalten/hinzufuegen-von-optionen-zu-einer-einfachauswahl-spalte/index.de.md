Einer Einfachauswahl-Spalte können Sie diverse **Optionen** hinzufügen, aus denen Sie im Anschluss beim Befüllen der Zellen wählen können. Grundsätzlich haben Sie **drei** verschiedene Möglichkeiten, Einfachauswahl-Optionen anzulegen.

- Optionen über die Spalteneinstellungen hinzufügen
- Optionen über die Zellen der Einfachauswahl-Spalte hinzufügen
- Optionen importieren

{{< warning  type="warning" headline="Mehrfachauswahl-Spalten"  text="Alle in diesem Artikel vorgestellten Funktionen gelten genauso für Mehrfachauswahl-Spalten." />}}

## Optionen über die Spalteneinstellungen hinzufügen

![Hinzufügen von Optionen zu einer Einfachauswahl-Spalte](images/add-options-to-a-single-select-culumn.gif)

1. Klicken Sie auf das **Dreieck-Symbol** rechts neben dem Namen einer beliebigen Einfachauswahl-Spalte.
2. Klicken Sie auf **Einfachauswahl bearbeiten**.
3. Fügen Sie der Einfachauswahl beliebig viele **Optionen** hinzu.
4. Bei Bedarf können Sie die den Optionen zugewiesenen **Farben** anpassen.
5. Bestätigen Sie jede Eingabe mit einem Klick auf **Option hinzufügen** oder mithilfe der **Enter-Taste**.

## Optionen über die Zellen der Einfachauswahl-Spalte hinzufügen

![Hinzufügen von Optionen über die Zeilen der Einfachauswahl-Spalte](images/Hinzufuegen-von-Optionen-ueber-die-Zeilen.png)

1. Klicken Sie zweimal in eine beliebige **Zelle** der Einfachauswahl-Spalte.
2. Geben Sie in das erschienene Suchfeld einen beliebigen **Wert** (Zahl, Begriff etc.) ein, den Sie als **neue Option** definieren möchten.
3. Klicken Sie auf **Option hinzufügen**.
4. Der eingegebene Wert wird anschließend der ausgewählten Zelle zugewiesen und zu den verschiedenen **Auswahl-Optionen** der Spalte hinzugefügt.

## Optionen importieren

Neben der **manuellen Eingabe** können Sie Optionen auch bequem aus einer bestehenden Einfachauswahl-Spalte exportieren und in eine andere Spalte dieses Typs **importieren**. Das ist sehr nützlich, wenn Sie immer wieder **die gleichen Optionen** über verschiedene Spalten hinweg verwenden möchten.

Mehr dazu erfahren Sie im Artikel [Optionen in Einfach- oder Mehrfach-Auswahlspalten exportieren und importieren](https://seatable.io/docs/auswahlspalten/einfachauswahl-optionen-exportieren-und-importieren/).

## Farben der Auswahl-Optionen

Um die Datenansicht innerhalb einer Tabelle übersichtlicher zu gestalten, ordnet SeaTable jeder Option in einer Einfachauswahl- oder Mehrfachauswahl-Spalte eine andere **Farbe** zu. Über die **Spalteneinstellungen** können Sie diese zufällig zugewiesene Farbe beliebig **anpassen**. Sie öffnen den **Farbwähler**, indem Sie auf das farbig hinterlegte **Dreieck-Symbol** vor dem Namen der Option klicken.

![Farben der Auswahl-Optionen](images/Farben-der-Optionen-einer-Einfachauswahlspalte-anpassen.png)

Mit einem [Enterprise-Abo](https://seatable.io/docs/teamverwaltung-abonnement/abo-pakete/#3-toc-title) haben Sie zudem die Möglichkeit, [eigene Farben](https://seatable.io/docs/arbeiten-mit-bases/eigene-farben-in-einer-base-hinzufuegen/) für Ihre Auswahl-Optionen anzulegen.
