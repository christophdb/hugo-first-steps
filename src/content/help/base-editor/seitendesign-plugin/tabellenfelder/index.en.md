**Table fields** contain the entries from one column of your table. Therefore, for the table fields, the [page design plugin](https://seatable.io/en/docs/seitendesign-plugin/anleitung-zum-seitendesign-plugin/) lists **all the columns of** the table from which you can include data in the page.

Since it is up to the number and type of columns which table fields you have to choose from, you have different options depending on the table. The content of the table fields inserted on a page depends on the **entries** in the table and changes **automatically** depending on row.

## Entries

An [entry](https://seatable.io/en/docs/seitendesign-plugin/eintraege-und-seiten/) in the page design plugin refers to one of the **row** of the selected table. When you switch between entries using the **Previous Entry** and **Next Entry** navigation options, data from other rows is automatically displayed in the table fields.

![Second entry of the used table for the page plugin.](https://seatable.io/wp-content/uploads/2022/11/Tabelle-2-eintraege.png)  
![Entry two when using the page plugin.](https://seatable.io/wp-content/uploads/2022/11/Eintraege-2.gif)

This makes it clear: Using table fields, you can quickly display information from the respective records in your template and create different versions of a document. You can also use the {{< seatable-icon icon="dtable-icon-open" >}} **Expand Entry** button to open the **row details** to edit the respective entry in the table.

## Application example

The **columns of** the "Budget Book" table should be displayed as **table fields** in the page design plugin.

![Table example Budget Book.](https://seatable.io/wp-content/uploads/2022/11/Tabelle-Budget-Book.png)

Create a **new page** in the plugin or open a **saved page design**. Read the [instructions for the page design plugin](https://seatable.io/en/docs/seitendesign-plugin/anleitung-zum-seitendesign-plugin/) to find out how to do this. The columns of the table are now available as table fields in the **page elements**.

![Table fields of the Budget Book table.](https://seatable.io/wp-content/uploads/2022/11/Tabellenfelder-Burdget-Book.png)

**Drag and drop** the table fields to the desired locations on the page. The table fields are **automatically** filled with the column information of an entry.

![Page design plugin example Budget Book.](https://seatable.io/wp-content/uploads/2022/11/Seitendesign-Plugin-Budget-Book.png)
