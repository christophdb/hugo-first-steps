**Поля таблицы** содержат записи из столбца вашей таблицы. Поэтому в [плагине дизайна страницы](https://seatable.io/ru/docs/seitendesign-plugin/anleitung-zum-seitendesign-plugin/) в полях таблицы перечислены **все столбцы** таблицы, данные из которых можно включить в страницу.

Поскольку от количества и типа столбцов зависит, какие поля таблицы вам нужно выбрать, у вас есть разные варианты в зависимости от таблицы. Содержимое полей таблицы, вставляемых на страницу, зависит от **записей** в таблице и **автоматически** изменяется в зависимости от строки.

## Записи

[Запись](https://seatable.io/ru/docs/seitendesign-plugin/eintraege-und-seiten/) относится к **строке в** выбранной таблице в плагине дизайна страницы. Если вы переключаетесь между записями с помощью навигационных опций **Предыдущая запись** и **Следующая запись**, данные из других строк автоматически отображаются в полях таблицы.

![Вторая запись таблицы, используемой для плагина страницы.](https://seatable.io/wp-content/uploads/2022/11/Tabelle-2-eintraege.png)  
![Ввод второй при использовании плагина страницы.](https://seatable.io/wp-content/uploads/2022/11/Eintraege-2.gif)

Таким образом, становится ясно: с помощью полей таблицы можно быстро отображать информацию из соответствующих записей в шаблоне и создавать различные версии документа. С помощью кнопки {{< seatable-icon icon="dtable-icon-open" >}} **Расширить запись** можно также открыть **детали строки для** редактирования соответствующей записи в таблице.

## Пример применения

**Колонки** таблицы "Бюджетная книга" должны отображаться как **поля таблицы** в плагине дизайна страницы.

![Таблица пример Бюджетная книга.](https://seatable.io/wp-content/uploads/2022/11/Tabelle-Budget-Book.png)

Создайте **новую страницу** в плагине или откройте **сохраненный дизайн страницы**. Как это сделать, читайте в [инструкции к плагину для дизайна страниц](https://seatable.io/ru/docs/seitendesign-plugin/anleitung-zum-seitendesign-plugin/). Столбцы таблицы теперь доступны как поля таблицы в **элементах страницы**.

![Поля таблицы бюджетной книги.](https://seatable.io/wp-content/uploads/2022/11/Tabellenfelder-Burdget-Book.png)

**Перетащите** поля таблицы в нужные места на странице. Поля таблицы **автоматически** заполняются информацией столбцов записи.

![Плагин для дизайна страницы Пример бюджетной книги.](https://seatable.io/wp-content/uploads/2022/11/Seitendesign-Plugin-Budget-Book.png)
