Com o [plugin de desenho da página](https://seatable.io/pt/docs/seitendesign-plugin/anleitung-zum-seitendesign-plugin/), é importante compreender a diferença entre as entradas e as páginas. Enquanto **páginas** adicionais estendem um documento independentemente do conteúdo da tabela, as **entradas** mapeiam registos diferentes das filas da tabela.

## Inscrições

Uma **entrada** refere-se a uma **linha da** tabela seleccionada no plug-in de design da página. Isto torna-se claro quando se utilizam [campos de tabela](https://seatable.io/pt/docs/seitendesign-plugin/tabellenfelder/) que recuperam dados das colunas de uma tabela: Quando se alterna entre entradas usando as opções de navegação **Entrada Anterior** e **Entrada Seguinte**, os campos da tabela exibem automaticamente dados de outras linhas.

Por exemplo, pretende inserir as entradas da coluna de texto "Nome" de uma tabela numa página para poder personalizar uma newsletter.

![Primeira entrada da tabela utilizada para o plugin de página.](https://seatable.io/wp-content/uploads/2022/11/Tabelle-1-eintraege.png)

Para tal, arraste o campo da tabela "Nome" para a página, de modo a que a informação da coluna do nome seja automaticamente mapeada para a página.

![Entrada numa das mesas em Setienplugin.](https://seatable.io/wp-content/uploads/2022/11/Eintraege-1.gif)

Ao utilizar os navegadores **Previous/Next Entry**, pode agora saltar entre as linhas para ter a informação apresentada alternadamente de todas as linhas da tabela no layout da página.

![Segunda entrada da tabela utilizada para o plugin de página.](https://seatable.io/wp-content/uploads/2022/11/Tabelle-2-eintraege.png)  
![Entrada dois ao utilizar o plug-in de página.](https://seatable.io/wp-content/uploads/2022/11/Eintraege-2.gif)

## Páginas

Pode adicionar **páginas** para expandir o documento e criar um **modelo com várias páginas**. Para o fazer, clicar no botão **Adicionar Página**.

![Inserir uma página no plugin de página.](https://seatable.io/wp-content/uploads/2022/11/Einfuegen-einer-Seite-in-dem-Seitenplugin.png)

**Clique com o botão direito do rato para** abrir um menu suspenso, que lhe oferece opções adicionais.

![O menu drop-down das páginas no plug-in da página.](https://seatable.io/wp-content/uploads/2022/11/Seitenplugin-Seiten-dropdown-menue.png)

Para **alternar** entre as páginas, basta clicar no **thumbnail** da página correspondente na barra de páginas. Pode alterar a **ordem das páginas** mantendo pressionado o botão do rato nos **seis pontos** {{< seatable-icon icon="dtable-icon-drag" >}} e **movendo** a miniatura da página.
