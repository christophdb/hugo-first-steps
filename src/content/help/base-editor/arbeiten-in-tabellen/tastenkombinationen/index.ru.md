В SeaTable повторяющиеся действия и операции можно удобно выполнять с помощью комбинаций клавиш. Вставка нового ряда, отображение сведений о ряде или отмена действия - в SeaTable это можно сделать за доли секунды с помощью правильной комбинации клавиш.

Если нужная комбинация клавиш не работает, возможно, вы используете другую раскладку клавиатуры. Подробности вы найдете в конце этой страницы." />}}

Кстати, чтобы получить быстрый обзор всех комбинаций клавиш, вам не нужно постоянно возвращаться на этот сайт. Вы также можете найти обзор всех комбинаций клавиш непосредственно в SeaTable. Откройте его с помощью {{< keyboard "Ctrl" >}} {{< keyboard "?" >}} или нажав на значок вопросительного знака в правом верхнем углу.

## Примечания

- При использовании комбинаций клавиш не имеет значения, будет ли буква типа {{< keyboard "F" >}} является верхним или нижним регистром. Чтобы нажать комбинацию клавиш, не обязательно переключаться на ввод букв верхнего регистра.
- Всегда нажимайте и удерживайте сначала функциональную клавишу, а затем буквенную. Например: Чтобы {{< keyboard "Ctrl + F" >}} сначала удерживайте нажатой кнопку {{< keyboard "Ctrl" >}} а затем нажмите один раз {{< keyboard "F" >}}.
