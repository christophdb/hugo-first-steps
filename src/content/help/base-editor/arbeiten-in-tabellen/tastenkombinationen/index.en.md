In SeaTable, recurring actions and actions can be conveniently performed using key combinations. Inserting a new row , displaying the row details or undoing an action - in SeaTable this can be done in a fraction of a second with the right key combination.

{{< warning  headline="⚠️ Observe keyboard layout"  text="The following table shows the shortcuts for an English keyboard layout. If the desired key combination does not work, it is probably because you are using a different keyboard layout. You can find details about this at the end of this page." />}}

By the way, to quickly get an overview of all keyboard shortcuts, you don't have to keep returning to this web page. You can also find the overview of all keyboard shortcuts directly in SeaTable. Open it with {{< keyboard "Ctrl" >}} {{< keyboard "?" >}} or by clicking on the question mark icon in the upper right corner.

![Keyboard shortcuts in SeaTable](https://seatable.io/wp-content/uploads/2022/08/keyboard-shortcuts.png)

## Notes

- For key combinations it does not matter whether a letter like {{< keyboard "F" >}} is uppercase or lowercase. To press a key combination, you do not have to switch to entering uppercase letters.
- Always press and hold the function key first and then the letter key. For example: To {{< keyboard "Ctrl + F" >}} first hold down the {{< keyboard "Ctrl" >}} and then press once {{< keyboard "F" >}}.

## Comparison table German/English keyboard layout

The labeling of function keys and the arrangement of some keys affected in this article are different on a German and an English keyboard. Since the table above is for English keyboards, below is a table comparing affected keys.

For example: If on an English keyboard the key combination {{< keyboard "Ctrl" >}} {{< keyboard ";" >}} should be pressed, you can use a German keyboard to {{< keyboard "Ctrl" >}} {{< keyboard "Ö" >}} press.

| -------------------------- | --------------------------- |
| {{< keyboard "?" >}}       | {{< keyboard "#" >}}        |
| {{< keyboard "Ctrl" >}}    | {{< keyboard "Ctrl" >}}     |
| {{< keyboard "Shift" >}}   | {{< keyboard "⇑" >}}        |
| {{< keyboard ";" >}}       | {{< keyboard "Ö" >}}        |
| {{< keyboard "'" >}}       | {{< keyboard "Ä" >}}        |
| {{< keyboard "Space" >}}   | {{< keyboard "Space" >}}    |
