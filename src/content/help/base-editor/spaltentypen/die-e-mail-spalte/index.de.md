Die E-Mail-Spalte ist ein Spaltentyp, mit dem Sie E-Mail-Adressen erfassen können. Im Unterschied zu einer Text-Spalte wird bei der E-Mail-Spalte sichergestellt, dass die Eingabe die Struktur einer gültigen E-Mail-Adresse hat.

Zusätzlich wird ein kleiner Briefumschlag eingeblendet, sobald man eine Zelle in der E-Mail-Spalte auswählt. Ein Klick auf dieses Symbol öffnet Ihr E-Mail-Programm, sodass Sie direkt eine E-Mail an die ausgewählte Adresse schreiben können.

## Verwendung der E-Mail-Spalte

1. Legen Sie über das **Plus-Symbol** eine neue Spalte mit dem Spaltentyp **E-Mail** an.
2. Geben Sie eine **gültige E-Mail-Adresse** in eine Zelle der Spalte ein.
3. Klicken Sie auf die Zelle, in der die E-Mail-Adresse steht, und ein **Briefumschlag** erscheint.
4. Klicken Sie auf das Symbol, um eine **neue E-Mail** in Ihrem Standard-E-Mail-Programm zu öffnen.
