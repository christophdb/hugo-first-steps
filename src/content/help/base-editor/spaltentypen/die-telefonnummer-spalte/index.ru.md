В SeaTable есть отдельный тип колонки для ввода **телефонных номеров**. По сравнению с обычным текстовым столбцом, столбец телефонных номеров имеет то преимущество, что Вы можете **вызвать телефонный номер** одним щелчком мыши.

## Создайте колонку телефонных номеров

![Создать колонку с телефонным номером](images/column-type-phone-number.gif)

1. Нажмите на символ **Плюс** справа от последнего столбца в заголовке таблицы.
2. Дайте столбцу **название**.
3. Выберите **тип столбца Номер телефона**.
4. Подтвердите выбор нажатием **Отправить**.

### Установите значение по умолчанию

Вы можете задать предустановленное [значение по умолчанию]({{< relref "help/base-editor/spalten/standardwert-fuer-eine-spalte-festlegen" >}}) для каждого столбца телефонных номеров. Это значение автоматически вводится в каждую новую строку таблицы.

### Валидация ввода

Как и в случае с текстовыми колонками, у Вас есть возможность проверить правильность ввода в колонке телефонных номеров. Выполнив **проверку формата**, Вы можете выделить телефонные номера, которые отклоняются от допустимого формата. Вы также можете использовать [регулярные выражения]({{< relref "help/base-editor/spaltentypen/validierung-der-eingabe-bei-textfeldern" >}}) для проверки.

![Проверка формата телефонного номера](images/format-check-phone-number.png)

## Стартовые вызовы

Тип колонки с номером телефона делает Ваше ежедневное общение еще проще. Щелчок по **значку телефон в ячейке** передает номер телефона в Ваше приложение телефонии. На мобильных телефонах это работает без каких-либо дополнительных настроек. То же самое можно сделать и на рабочем столе, если в браузере был создан ярлык для программы телефонии.
