Die **Reihenfolge**, in der Ihre Ansichten einer Tabelle aufgelistet sind, können Sie beliebig ändern. Das Verschieben erfolgt bequem per **Drag-and-Drop**.

## Reihenfolge der Ansichten verändern

![Die Reihenfolge von Ansichten ändern.](images/Die-Reihenfolge-von-Ansichten-aendern.gif)

1. Klicken Sie auf den **Namen der aktuellen Ansicht**.
2. Bewegen Sie den Mauszeiger auf die Ansicht, die Sie verschieben wollen.
3. Halten Sie die linke Maustaste auf der **Sechs-Punkte-Greiffläche** gedrückt und verschieben Sie die Ansicht. 
4. Eine **schwarze Linie** zeigt Ihnen die Position, an die die Ansichtszeile springen wird. Lassen Sie die Maustaste los, sobald die Ansicht an der gewünschten Position ist.

## Häufige Fragen

{{< faq "Ist die Reihenfolge für jeden Benutzer individuell?" >}}
Nein. Die Reihenfolge der Ansichten ist für alle Benutzer einer Base die gleiche.
{{< /faq >}}

{{< faq "Können Ansichten auch gruppiert werden?" >}}
Natürlich. Ansichten lassen sich ganz einfach in [Ordnern]({{< relref "help/base-editor/ansichten/ansichten-in-ordnern-gruppieren" >}}) organisieren.
{{< /faq >}}

{{< faq "Verhalten sich private Ansichten anders?" >}}
Nein. Private Ansichten können Sie genauso per Drag-and-Drop verschieben. Private Ansichten sind aber nur für Sie sichtbar.
{{< /faq >}}
