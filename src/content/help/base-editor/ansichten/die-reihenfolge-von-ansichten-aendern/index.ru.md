Вы можете изменить **порядок** перечисления представлений таблицы по своему усмотрению. Перемещение осуществляется удобным способом **перетаскивания**.

## Изменение порядка представления

1. Щелкните по **названию текущего представления**.
2. Подведите мышь к виду, который нужно переместить.
3. Удерживать появление **Символ с шестью точками**

    ![Иконка для перемещения элементов](images/move-icon.png)
    для перемещения и перетаскивания вида в нужное положение. A **чёрная линия** показывает позицию, в которую перейдет линия.

4. Отпустите кнопку мыши, как только вид окажется в нужном положении.

![Измените порядок просмотра.](images/Die-Reihenfolge-von-Ansichten-aendern.gif)

## Часто задаваемые вопросы

{{< faq "Является ли заказ индивидуальным для каждого пользователя?" >}}Нет. Порядок просмотра одинаков для всех пользователей базы.
{{< /faq >}}
{{< faq "Можно ли также группировать представления?" >}}Конечно. Представления можно легко организовать в [папки]({{< relref "help/base-editor/ansichten/ansichten-in-ordnern-gruppieren" >}}).
{{< /faq >}}
{{< faq "Ведут ли себя частные взгляды по-другому?" >}}Нет. Вы можете перемещать частные представления таким же образом, используя перетаскивание. Однако частные виды видны только вам.

{{< /faq >}}
