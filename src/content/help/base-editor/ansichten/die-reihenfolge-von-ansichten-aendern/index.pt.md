Pode alterar a **ordem** em que as suas vistas de uma tabela são listadas como desejar. A deslocação é convenientemente efectuada através de **arrastar e largar**.

## Alterar a ordem de vistas

![Alterar a ordem das vistas](images/Die-Reihenfolge-von-Ansichten-aendern.gif)

1. Clique sobre o **nome da vista actual**.
2. Mova o rato para a vista que pretende mover.
3. Mantenha premido o botão esquerdo do rato sobre a **superfície de preensão de seis pontos** e desloque a vista.  
4. A **linha preta** mostra-lhe a posição para a qual a linha irá saltar. Solte o botão do rato assim que a vista estiver na posição desejada.

## Perguntas mais frequentes

{{< faq "A ordem de vistas é individual para cada utilizador?" >}}
Não. A ordem de vistas é a mesma para todos os utilizadores de uma base.
{{< /faq >}}
{{< faq "As vistas também podem ser agrupadas?" >}}
Claro que sim. As vistas podem ser facilmente organizadas em [pastas]({{< relref "help/base-editor/ansichten/ansichten-in-ordnern-gruppieren" >}}).
{{< /faq >}}
{{< faq "As vistas privadas comportam-se de forma diferente?" >}}Não. Pode-se mover vistas privadas da mesma forma usando o arrastar e largar. No entanto, as vistas privadas só são visíveis para si.
{{< /faq >}}
