SeaTable предлагает различные методы анализа и подготовки данных. **Фильтры, сортировка и группировка** - это простые инструменты, с помощью которых Вы можете структурировать данные всего за несколько кликов и быстро получить нужную информацию. При правильном использовании фильтры, сортировка и группировка предоставляют широкий спектр возможностей для анализа.

Записи данных фильтруются, сортируются и группируются с помощью **правил**. Правило всегда состоит из инструкции и столбца, к которому оно применяется. В случае сортировки и группировки инструкция - это порядок (по возрастанию или по убыванию), в котором перечислены записи данных, независимо от типа столбца. Инструкция фильтра имеет дополнительные компоненты, которые зависят от типа столбца.

Звучит сложно? Вовсе нет – группировка, сортировка и фильтрация в SeaTable - это детская забава!

## Группировка

Функция группировки позволяет Вам **собрать строки таблицы в группы** и вычислить простую описательную статистику, например, количество строк в группе или сумму и среднее значение столбцов, основанных на числах. Группировка производится с помощью **одинаковых значений в столбце, выбранном для группировки**. Например, организуйте задачи в Вашей команде, сгруппировав невыполненные задачи для каждого члена команды с помощью столбца "Сотрудник".

Если Вы хотите отобразить данные еще более точно, используйте **вложенные группировки**. Вы можете выбрать до трех характеристик группировки, чтобы создать подчиненные группировки. Например, Вы можете группировать задачи сначала по ответственному лицу, затем по приоритету, а потом по дате. Это открывает новые возможности для структурирования данных.

## Сортировка

Вы можете использовать функцию сортировки, чтобы отобразить неупорядоченные записи в таблице **в упорядоченной последовательности**.

Создайте правило для сортировки записей данных по возрастанию или убыванию в соответствии со значениями в столбце. **Несколько правил сортировки** также можно использовать для больших записей данных. Если используется несколько правил сортировки, то сначала строки сортируются в соответствии с первым правилом. Строки с одинаковыми значениями по первому правилу сортировки затем сортируются по второму правилу.

[Подробнее о функции сортировки здесь]({{< relref "help/base-editor/ansichten/sortieren-von-eintraegen-in-einer-ansicht" >}})

## Фильтр

С помощью функции фильтрации SeaTable Вы можете использовать правила фильтрации, чтобы **отфильтровать** определенные записи данных из таблицы и отобразить только те строки, которые соответствуют нужному критерию или критериям. Это позволяет Вам сократить количество строк в представлении до необходимого минимума и всегда иметь обзор.

В отличие от правил группировки и сортировки, инструкция для правил фильтрации обычно состоит из двух компонентов:
- **Критерий фильтра**: Связь между фильтруемым столбцом и значением фильтра (например, "есть", "содержит", "пустой")
- **Значение фильтра**: Значение, по которому будет производиться фильтрация (например, "123", "сделано")

Правила фильтрации также могут быть связаны с помощью двух булевых операторов AND и OR. При использовании ссылки **AND** отображаются только те записи, которые удовлетворяют всем заданным условиям. Ссылка **OR** может быть использована для фильтрации представления таким образом, чтобы отображались все строки, удовлетворяющие хотя бы одному из заданных условий. Операторы AND и OR также можно использовать вместе, используя **группы фильтров**.

## Блокировка настроек фильтра, сортировки и группировки

Настройки, которые Вы сделали с помощью фильтров, группировки и сортировки, всегда сохраняются в текущем представлении. Если Вы хотите сохранить их для дальнейшего использования, Вы можете заблокировать [вид]({{< relref "help/base-editor/ansichten/ansicht-sperren" >}}), чтобы никто не мог изменить вид.
