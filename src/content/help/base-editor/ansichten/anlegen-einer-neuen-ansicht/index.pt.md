No SeaTable, pode criar diferentes vistas para as suas tabelas e, assim, visualizar os mesmos dados de diferentes formas. Por exemplo, se pretender filtrar um valor específico numa tabela, pode guardar a vista filtrada da tabela e, assim, aceder-lhe mais rapidamente no futuro sem ter de filtrar novamente.

## Para criar uma nova vista no SeaTable

![Crie uma nova vista](images/create-table-view.jpg)

1. Clique no **nome da vista atual**.
2. Clique em **Adicionar vista ou pasta** e selecione o **tipo de vista** pretendido.

![Dê um nome a uma vista](images/name-table-view.jpg)

3. Dê um **nome** à nova vista.
4. Active o cursor se a nova vista não deve ser visível para todos, mas **privada**.
5. Confirme com **Submeter**.

A **vista** recém-criada abre-se automaticamente e pode **filtrar**, **ordenar** ou **agrupar** de acordo com quaisquer valores, por exemplo.

{{< warning  headline="Dica" >}}

Informações detalhadas sobre as três ferramentas, **agrupamento**, **classificação** e **filtragem**, podem ser encontradas [aqui]({{< relref "help/base-editor/ansichten/gruppierung-sortierung-und-filter" >}}).

{{< /warning >}}

Além disso, tem a opção de criar vistas privadas. Pode descobrir como as vistas privadas diferem das vistas normais nos seguintes pontos [neste artigo]({{< relref "help/base-editor/ansichten/unterschiede-zwischen-privaten-und-normalen-ansichten" >}}).
