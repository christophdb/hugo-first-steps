Vous pouvez supprimer à tout moment les vues créées à titre de test ou devenues non pertinentes en sélectionnant l'option **Supprimer la vue**.

## Supprimer une vue

1. Cliquez sur le **nom de la vue actuelle**.
2. Passez la souris sur la vue que vous souhaitez supprimer.
3. **Trois points** sont visibles dans la ligne de la vue. Sélectionnez-les.
4. Sélectionnez l'option **Supprimer la vue**.

![Supprimer une vue](images/delete-view.jpg)

## Questions fréquentes

{{< faq "Est-il possible de restaurer une vue supprimée ?" >}}**Non**. Étant donné qu'une vue ne contient pas de données propres, mais qu'il s'agit simplement d'une représentation spéciale des données, il n'est pas possible de restaurer une vue supprimée via la Corbeille ni via le raccourci {{< key "CTRL" >}}+{{< key "Z" >}}.
{{< /faq >}}
{{< faq "Une vue peut-elle toujours être supprimée ?" >}}Vous ne pouvez supprimer des vues que si vous avez créé **au moins une vue supplémentaire** à la Default View.
{{< /faq >}}
