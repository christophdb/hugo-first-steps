You can delete test views or views that have become irrelevant at any time by selecting the **Delete view** option.

## Delete a view

1. Click the **name of the current view**.
2. Hover the mouse over the view you want to delete.
3. **Three points** become visible in the view line. Select them.
4. Select the **Delete view** option.

![Delete a view](images/delete-view.jpg)

## Frequently asked questions

{{< faq "Can a deleted view be restored?" >}}
**No**. Since a view does not contain any data of its own, but is merely a special representation of the data, you cannot restore a deleted view using either the trash or the shortcut {{< key "CTRL" >}}+{{< key "Z" >}}.
{{< /faq >}}

{{< faq "Can a view always be deleted?" >}}You can delete views only if you have created **at least one additional view** to the **default view**.
{{< /faq >}}
