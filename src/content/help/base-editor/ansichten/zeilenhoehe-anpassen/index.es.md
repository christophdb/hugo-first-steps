Por defecto, SeaTable muestra todas las filas con una altura de fila compacta para mostrar el mayor número posible de registros. Sin embargo, con esta altura **de** fila simple, entre otras cosas, se cortan las [columnas de]({{< relref "help/base-editor/spalten/wie-man-tabellen-in-seatable-miteinander-verknuepft" >}}) y [selección]({{< relref "help/base-editor/spalten/die-mehrfachauswahl-spalte" >}}) múltiple y solo se muestran pequeñas **miniaturas** de imágenes.

![Contenido de la celda truncado](images/small-row-height-cut-cells.png)

Esta altura de fila puede ser adecuada para la mayoría de sus casos de uso, pero puede haber situaciones en las que desee ver más contenido de las filas respectivas. Por eso SeaTable le permite ajustar la altura de las filas en cada una de sus vistas.

Para ello se dispone de cuatro alturas de línea diferentes:

- Simple (el valor por defecto para las nuevas tablas y vistas)
- Doble
- Triple
- Cuádruple

## Ajustar la altura de la fila

![Ajustar la altura de la fila](images/set-row-height-new.gif)

1. Haga clic en el icono de **altura de fila** en las opciones de vista.
2. **Seleccione** la altura de fila deseada.
   {{< warning  headline="Una altura de línea para todas las líneas"  text="La altura de la fila se ajusta siempre para **todas las** filas de una vista. No es posible realizar alturas de fila individuales dentro de una vista." />}}
