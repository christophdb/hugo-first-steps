In jeder Tabelle haben Sie die Möglichkeit, alle verfügbaren [Ansichten]({{< relref "help/base-editor/ansichten/was-ist-eine-ansicht" >}}) dauerhaft in einer **Navigationsleiste** am linken Rand einzublenden. Das ergibt vor allem Sinn, wenn Sie oft zwischen Ansichten wechseln und ein **Schnellzugriff** dies effizienter macht.

Die angepinnte Ansichtsnavigation bleibt auch sichtbar, wenn Sie innerhalb einer Tabelle nach rechts scrollen. So können Sie problemlos Einträge in verschiedenen Ansichten miteinander vergleichen.

## So blenden Sie die Ansichtsnavigation ein und aus

1. Klicken Sie auf den **Namen der aktuellen Ansicht**.
2. Klicken Sie auf die **Pin-Nadel** {{< seatable-icon icon="dtable-icon-pin" color="grey" >}} in der oberen rechten Ecke.

![Anpinnen der verschiedenen Ansichten einer Tabelle](images/pin-view-list.gif)

Solange die Pin-Nadel **rot** markiert ist, sind die verfügbaren Ansichten am linken Seitenrand **angepinnt**. Um die Ansichtsnavigation wieder **auszublenden**, klicken Sie erneut auf die Pin-Nadel.
