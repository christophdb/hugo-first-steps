Die **Dauer-Spalte** eignet sich besonders gut, um die Dauer von verschiedenen Ereignissen festzuhalten. Sie kann dabei einen **Zeitraum** auf die Minute oder gar auf die Sekunde genau erfassen. Beispielsweise bietet es sich für die Arbeitszeiterfassung an, die Dauer-Spalte in Verbindung mit einer [Datum-Spalte](https://seatable.io/docs/datum-dauer-und-personen/die-datum-spalte/) zu nutzen.

![Anwendungsbeispiel für eine Dauer-Spalte](https://seatable.io/wp-content/uploads/2023/01/Die-Dauer-Spalte-1.png)

## Anlegen der Dauer-Spalte

Beim Anlegen der Dauer-Spalte können Sie entscheiden, ob Sie die Dauer auf die **Minute** oder gar auf die **Sekunde** genau erheben möchten.
