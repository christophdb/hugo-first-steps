Порядок появления опций в колонке с одним выбором можно изменить впоследствии.

{{< warning  headline="Несколько столбцов выбора"  text="Эта инструкция по щелчку работает аналогичным образом для нескольких столбцов выбора." />}}

## Простое расположение опций с помощью перетаскивания

![Изменение порядка опций одиночного выбора](images/change-the-order-of-single-select-options.gif)

1. Нажмите на **символ треугольника** справа от названия любого столбца с одиночным выбором.
2. Перейдите к **Редактировать отдельный выбор**.
3. **Перетащите** параметр в нужное место. Для этого используйте область с **шестью точками** слева от названия каждой опции.

## Сортировка и группировка

Порядок опций имеет решающее значение при [сортировке]({{< relref "help/base-editor/ansichten/sortieren-von-eintraegen-in-einer-ansicht" >}}) или [группировке]({{< relref "help/base-editor/ansichten/gruppieren-von-eintraegen-in-einer-ansicht" >}}) по одному столбцу выбора. Вы можете гибко организовать свои представления, расположив опции по своему усмотрению.

![Изменение сортировки одного столбца выбора](images/Sortierung-einer-Einfachauswahl-Spalte-aendern.gif)
