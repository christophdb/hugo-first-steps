Чтобы предотвратить нежелательные изменения строк, можно **заблокировать строки**. Это может быть полезно, например, если в таблице работает более одного человека. Блокировка может осуществляться как **вручную** вами, так и **автоматически** с помощью [автоматики](https://seatable.io/ru/docs/arbeiten-mit-automationen/zeilen-per-automation-sperren/).

## Чтобы заблокировать ряд

1. Откройте любую таблицу, в которой вы хотите заблокировать **строку**.
2. **Щелкните правой кнопкой мыши** на строке, чтобы открыть выпадающее меню.
3. Выберите опцию **Lock Row (Заблокировать строку** ).

Как только строка блокируется, в **колонке нумерации** появляется маленький **красный треугольник**.

![Блокировка линии](https://seatable.io/wp-content/uploads/2022/10/sperren-einer-zeile-1.png)

## Разблокировать ряды

Так же, как вы заблокировали ряд, вы можете снова его **разблокировать**. Для этого выполните описанные выше инструкции по щелчку и выберите опцию **Unlock Row**.

{{< warning  headline="Требуется разрешение"  text="Вы можете разблокировать только те строки, которые заблокировали **сами** - если у вас нет **прав администратора**. Затем вы также можете разблокировать строки, которые были заблокированы **другими** или **автоматикой**." />}}

## Фиксация рядов с помощью автоматики

{{< required-version "Предприятие" >}}

В SeaTable вы можете создавать **правила автоматизации** для **автоматической блокировки строк**, как только они удовлетворяют определенным условиям.

Это очень полезно, если вы хотите заблокировать много строк одновременно или всегда хотите предотвратить дальнейшее редактирование строки с определенного момента времени (например, когда процесс завершен).

**Подробнее об этом вы можете узнать в статье** [Блокировка строк с помощью автоматизации](https://seatable.io/ru/docs/arbeiten-mit-automationen/zeilen-per-automation-sperren/).

![Блокировка линии с помощью автоматики](https://seatable.io/wp-content/uploads/2022/10/lock-row-with-an-automation.png)

{{< warning  headline="Разблокировка возможна только вручную"  text="Вы **не можете** разблокировать строки, заблокированные с помощью автоматизации, таким же образом. Это возможно только вручную и с необходимыми **правами администратора**." />}}
