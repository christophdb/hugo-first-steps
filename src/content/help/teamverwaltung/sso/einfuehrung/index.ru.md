С выходом [SeaTable 6.0 в ноябре 2025 года]({{< relref "posts/seatable-releases/seatable-release-6-0" >}}) команды с подпиской Enterprise получают доступ к мощной новой функции: **Единый вход (SSO)**.

Эта функция позволяет командам входить в SeaTable Cloud значительно проще и безопаснее.

## Что такое SSO?

Единый вход (Single Sign-On, SSO) — это метод аутентификации, при котором пользователи заходят один раз в центральную систему — поставщика идентификации (IdP), а затем автоматически получают доступ к нескольким связанным сервисам, в данном случае к SeaTable Cloud. Ваша команда получает преимущества в виде меньшего количества проблем с паролями, улучшенной безопасности и упрощенного управления пользователями.

![Страница входа в SeaTable Cloud с входом через единый вход](images/login-with-sso.png)

## Три основных преимущества единого входа

### Меньше проблем с паролями

С SSO пользователям нужно запоминать только один пароль — от их поставщика идентификации (IdP). Это снижает количество учетных данных, которые необходимо управлять. Это уменьшает усталость от паролей — разочарование, вызванное необходимостью запоминать множество паролей. Также предотвращает ошибки, вызванные забытыми паролями или небезопасными практиками, такими как запись или повторное использование паролей.

### Повышенная безопасность

SSO повышает безопасность за счет централизации входа через IdP. Там можно применять современные методы защиты, например многофакторную аутентификацию. Учетные данные больше не хранятся отдельно для каждого приложения, что уменьшает поверхность атаки. Связь между IdP и SeaTable обеспечивает надежную аутентификацию и снижает риск использования слабых паролей.

### Упрощенное управление пользователями

SSO централизует управление пользователями, что значительно упрощает процесс. Администраторы команды удобно управляют учетными записями пользователей и разрешениями через IdP. Новые сотрудники легко добавляются, а процессы приема и увольнения проходят эффективнее. Автоматическое объединение учетных записей пользователей при первом входе через SSO также предотвращает дублирование и экономит усилия.

## Кто может использовать SSO?

Функция SSO доступна исключительно для **команд с подпиской Enterprise**. Каждый клиент Enterprise может использовать своего собственного IdP, что обеспечивает максимальную гибкость для интеграции в существующую корпоративную инфраструктуру.

## Как настроить

Настройка осуществляется администратором команды в управлении командами SeaTable Cloud. Сначала в IdP необходимо создать приложение и соответствующие настройки. Этот шаг технически прост и поддерживается большинством современных поставщиков идентификаций.

Другим важным требованием является подтверждение владения доменом через TXT-запись в DNS соответствующего домена электронной почты. Только после успешной верификации домен официально связывается с командой SeaTable.

### Что означает аутентификация домена?

После верификации домена все пользователи с электронной почтой этого домена могут входить через SSO. Это создает четкое разделение и автоматическое назначение пользователей команде.

### Объединение существующих учетных записей

Особенность — автоматическое связывание существующих учетных записей пользователей с входом через SSO при первом входе. Данные и настройки не теряются, учетные записи объединяются. Это значительно облегчает переход.

### Гибкие варианты входа

По умолчанию пользователи могут входить либо с прежним именем пользователя и паролем, либо с новым SSO после настройки. Это обеспечивает плавный переход. Для повышения безопасности администратор команды может в любой момент заставить использовать только вход через SSO.

## Основные преимущества

- Значительно упрощенный вход для всех участников команды
- Повышенная безопасность благодаря централизованной аутентификации
- Гибкая интеграция с собственным IdP
- Автоматическое объединение учетных записей в процессе перехода
- Улучшенное управление и контроль со стороны администраторов команды

Используя SSO в SeaTable Cloud, вы получаете современный контроль доступа, который совмещает удобство и безопасность, адаптированные под нужды современных предприятий.
