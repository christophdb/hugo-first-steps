В администрировании команды вы, как **администратор команды**, можете в любой момент внести изменения в подписку. В этой статье описано, как вносить изменения, когда они вступают в силу и как влияют на продолжительность.

## Эффективность изменений

В основном, **обновление** подписки происходит немедленно, в то время как **понижение** всегда происходит в конце текущего срока. Ниже приведена таблица, в которой вы можете в любой момент посмотреть **оставшийся срок** действия вашей подписки на SeaTable в **командной администрации**.

| Действие                          | Эффективность |
| --------------------------------- | ------------- |
| Бесплатно → Плюс                  | Сразу же      |
| Бесплатно → Предприятие           | Сразу же      |
| Плюс → Бесплатно                  | В конце срока |
| Предприятие → Бесплатно           | В конце срока |
| Больше членов команды             | Сразу же      |
| Меньшее количество членов команды | В конце срока |
| Ежемесячный → ежегодный биллинг   | Сразу же      |
| Годовой → ежемесячный биллинг     | В конце срока |

## Срок и пропорциональная оплата изменений

Изменения в подписке обычно не влияют на ее срок. Единственным исключением является переход с ежемесячной на ежегодную оплату. В этом случае с момента изменения начинается новый срок.

| Изменить                                   | Влияние на время работы |
| ------------------------------------------ | ----------------------- |
| Количество членов команды                  | нет                     |
| Обновление подписки                        | нет                     |
| Переход с ежемесячного на ежегодный платеж | новый срок              |

Разумеется, за каждое изменение вы платите только соответствующую **пропорциональную сумму**. Например, если вы добавляете еще одного члена команды, вы платите только пропорциональную стоимость в зависимости от оставшегося срока. При обновлении подписки из новой стоимости будет вычтена оставшаяся стоимость текущей подписки.

## Настроить текущую подписку

{{< warning  headline="Важное замечание"  text="Пакет подписки вашей команды может быть изменен только **администраторами команды**." />}}

1. Откройте **администрацию команды**.
2. Перейдите в раздел **Подписка**.
3. Нажмите кнопку **Настроить подписку**.
4. Внесите необходимые изменения в **подписку**, **количество пользователей** и **расчетный период**.
5. Подтвердите изменение подписки с помощью платной **подписки Adjust**.

![Подтвердите изменение вашей подписки](https://seatable.io/wp-content/uploads/2023/01/switch-from-plus-to-enterprise.png)

## Отмена платной подписки

Вы можете вернуться к бесплатной подписке по истечении уже оплаченного срока. Для этого необходимо отменить платную подписку.

{{< warning  headline="Расторжение договора не является отменой"  text="Отмена платной подписки не приводит к удалению вашей команды или данных. Ваша команда будет переведена на бесплатную версию SeaTable Free только по окончании срока действия подписки. О том, как полностью удалить свою команду, вы можете узнать [здесь](https://seatable.io/ru/docs/teamverwaltung/das-komplette-team-loeschen/)." />}}

1. Откройте **администрацию команды**.
2. Нажмите на **Подписку**.
3. Нажмите кнопку **Отменить подписку и перейдите на SeaTable Free**.

![Изменить подписку с Plus/Enterprise на Free](https://seatable.io/wp-content/uploads/2023/01/switch-to-seatable-free.png)

5. Внимательно прочитайте **инструкции по** переходу на SeaTable Free.
6. Активируйте **слайдер** и подтвердите изменение подписки командой **Отменить подписку и перейти на SeaTable Free**.

![Подтвердите изменение вашей подписки на SeaTable Free](https://seatable.io/wp-content/uploads/2023/01/confirm-switch-to-seatable-free.png)
