После завершения работы над сервисом вы хотите завершить процесс, отправив участникам заключительное письмо. Это письмо может иметь различное содержание, например

- Подтверждение участия
- Спасибо за участие
- Запрос на обратную связь
- Код ваучера на следующий курс
- …

## Создайте и отправьте подтверждение участия

В этом разделе мы будем использовать SeaTable для создания подтверждения участия в формате PDF и отправки его участникам по электронной почте.

### Объединение писем с помощью плагина дизайна страниц

SeaTable предлагает несколько плагинов, которые позволяют добавлять дополнительные представления и функции к вашим данным. Плагин дизайна страниц позволяет создавать PDF-документы - аналогично почтовому слиянию в Microsoft Word.

С помощью этого плагина вы можете создавать шаблоны, содержащие как статический контент (например, текст или логотипы), так и динамические данные из вашей базы. Каждая строка таблицы может генерировать отдельный документ.

{{< warning  headline="Универсальные возможности применения"  text="Плагин для оформления страниц предлагает множество возможностей: Помимо сертификатов участия, вы можете создавать счета-фактуры, сертификаты, циркуляры и многое другое." />}}

Мы уже подготовили простой шаблон для этого курса, чтобы создать подтверждение участия. Название, курс и дата проведения курса вставляются автоматически, а остальной текст остается неизменным.

Потратьте несколько минут на изучение возможностей плагина. Например, попробуйте добавить логотип или графику в свою подпись.

Если вы нажмете на `{{< seatable-icon icon="dtable-icon-download" >}} PDF` созданный PDF будет сохранен на вашем компьютере.

### Отправка по электронной почте

Конечно, вы не хотите скачивать PDF вручную и загружать его снова, чтобы отправить по электронной почте. Вместо этого вы можете нажать кнопку в окне `Registrations` который последовательно выполняет два действия:

- `{{< seatable-icon icon="dtable-icon-pdf" >}} Create PDF and save to column`
- `{{< seatable-icon icon="dtable-icon-email" >}} Send email`

Конфигурация этих действий должна быть уже знакома. При создании PDF выберите нужный шаблон и столбец, в котором будет сохранен документ. При отправке письма укажите столбец, из которого будет отправлено вложение.

{{< warning  headline="Действия выполняются одно за другим"  text="Не проблема, что кнопка сначала создает PDF-файл, а затем отправляет его. SeaTable ждет, пока PDF-файл будет создан, прежде чем отправить его по электронной почте." />}}

### Несколько слов об управлении файлами

В SeaTable файлы и изображения хранятся не непосредственно в ячейках, а в центральном хранилище данных базы. Ячейки содержат только ссылки на соответствующие файлы. Такое централизованное управление имеет ряд преимуществ:

- Файлы можно использовать несколько раз, не загружая их снова.
- Даже если вы удалите файл из ячейки, он останется доступным в центральном администрировании до тех пор, пока вы не удалите его окончательно.

Потратьте несколько минут, чтобы ознакомиться с управлением файлами в SeaTable. Дополнительную информацию можно найти в справочных статьях по ссылкам в конце этого раздела.
